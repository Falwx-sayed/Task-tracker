# Task Tracker CLI

A simple command-line interface (CLI) task manager built with Python.  
Track tasks you need to do, mark them as in-progress or done, update, delete, and list tasks by status.

---

## Features

- Add a new task
- List all tasks
- List tasks by status: `todo`, `in-progress`, `done`
- Update a task description
- Delete a task
- Mark a task as **in-progress**
- Mark a task as **done**

---

## Installation

1. Clone the repository:

```bash
git clone <YOUR_GITHUB_REPO_URL>
cd task-tracker


chmod +x task_cli.py
```
